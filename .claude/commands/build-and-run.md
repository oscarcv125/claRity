Build and run ClaRity on the iPhone 17 Pro simulator using XcodeBuildMCP.
1. Clean build folder
2. Build for iPhone 17 Pro, iOS 18
3. If build succeeds, launch on simulator
4. Report any errors with file:line references
5. If errors exist, fix them and rebuild

For device testing, use the physical iPad Air 13" M3 connected via Xcode.
