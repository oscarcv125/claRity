# claRity (DislexIA)

An iOS application designed to assist users with dyslexia.

## Overview
claRity (internally known as DislexIA) is a native iOS application built to provide accessibility tools and support for dyslexic readers.

## Development
- Built natively using **Xcode** (Swift).
- Includes comprehensive Unit Tests (`dislexiaTests`) and UI Tests (`dislexiaUITests`).
- Refer to `DislexIA_Dev_Spec.md` for detailed development specifications.
